BAB 3
ROLE DAN HAK AKSES

3.1. Penjelasan Role Pengguna

Aplikasi ini memiliki 8 (delapan) level pengguna dengan hak akses yang berbeda sesuai dengan tugas dan tanggung jawab masing-masing:

1. Admin
2. Guru BK (Bimbingan Konseling)
3. Wali Kelas
4. Guru
5. Kesiswaan
6. Kepala Sekolah
7. Siswa
8. Orang Tua

3.2. Hak Akses Setiap Role

3.2.1. Admin

Hak Akses:
- Full akses ke semua fitur aplikasi
- Kelola data master (siswa, guru, orang tua)
- Kelola jenis pelanggaran dan poin
- Kelola jenis prestasi dan poin
- Kelola jenis sanksi
- Verifikasi user baru (approve/reject)
- Input pelanggaran dan prestasi
- Backup dan restore database
- Lihat semua laporan dan statistik

Menu yang Tersedia:
- Dashboard
- Data Siswa
- Data Guru
- Data Orang Tua
- Jenis Pelanggaran
- Pelanggaran
- Prestasi
- Sanksi
- Verifikasi User
- Verifikasi Laporan
- Backup Database
- Laporan

3.2.2. Guru BK (Bimbingan Konseling)

- Input hasil bimbingan
- Kirim notifikasi ke orang tua
- Lihat riwayat pelanggaran siswa
- Lihat riwayat bimbingan
- Cetak laporan BK

Menu yang Tersedia:

- Input Pelanggaran
- Bimbingan Konseling

- Riwayat Bimbingan
- Laporan BK

3.4. Alur Verifikasi Pengguna Baru

Proses registrasi hingga akun aktif:

1. User melakukan registrasi di halaman register
   - Pilih jenis akun (Siswa/Wali Kelas/Orang Tua)
   - Isi form registrasi
   - Submit

2. Sistem mengirim email verifikasi
   - User cek inbox email
   - Klik link verifikasi di email
   - Email terverifikasi

3. Admin melakukan verifikasi akun
   - Admin login ke aplikasi
   - Buka menu "Verifikasi User"
   - Lihat daftar user yang perlu diverifikasi
   - Approve atau Reject akun

4. User menerima notifikasi
   - Jika Approved: User bisa login
   - Jika Rejected: User tidak bisa login

5. User login dengan akun yang sudah diverifikasi
   - Masuk ke dashboard sesuai role
   - Mulai menggunakan aplikasi

Catatan Penting:
- Akun harus melalui 2 tahap verifikasi (email + admin)
- Proses verifikasi biasanya 1-2 hari kerja
- Hubungi admin jika verifikasi lebih dari 3 hari

3.5. Cara Mengetahui Role Anda

Setelah login, Anda dapat mengetahui role Anda dengan cara:

1. Lihat di pojok kanan atas (nama dan role)
2. Lihat menu yang tersedia (setiap role punya menu berbeda)
3. Lihat dashboard (setiap role punya dashboard berbeda)

Contoh:
- Jika Anda melihat menu "Kelola Data Siswa" → Anda adalah Admin
- Jika Anda melihat menu "Bimbingan Konseling" → Anda adalah Guru BK
- Jika Anda melihat menu "Data Siswa Kelas" → Anda adalah Wali Kelas
- Jika Anda melihat menu "Verifikasi Pelanggaran" → Anda adalah Kesiswaan
- Jika Anda hanya bisa lihat data sendiri → Anda adalah Siswa
- Jika Anda melihat data anak → Anda adalah Orang Tua

3.6. Pergantian Role

Apakah bisa ganti role?

Tidak. Role ditentukan saat registrasi dan tidak bisa diubah sendiri oleh user.

Jika perlu ganti role:
1. Hubungi Admin
2. Admin akan mengubah role Anda di database
3. Logout dan login kembali
4. Role sudah berubah

Contoh kasus pergantian role:
- Guru yang menjadi Wali Kelas → role berubah dari Guru ke Wali Kelas
- Guru BK yang menjadi Wali Kelas → role berubah dari Guru BK ke Wali Kelas
- Siswa yang lulus → akun dinonaktifkan
- Guru yang pensiun → akun dinonaktifkan

3.7. Keamanan Akun Berdasarkan Role

Tips keamanan untuk setiap role:

Admin:
- Jangan bagikan password ke siapapun
- Ganti password secara berkala (minimal 1 bulan sekali)
- Selalu logout setelah selesai
- Jangan login di komputer umum

Guru (BK, Wali Kelas, Guru):
- Jangan biarkan komputer tanpa pengawasan saat login
- Logout jika meninggalkan komputer
- Jangan simpan password di browser

Kesiswaan & Kepala Sekolah:
- Password harus kuat (kombinasi huruf, angka, simbol)
- Jangan login di perangkat orang lain
- Selalu logout setelah selesai

Siswa & Orang Tua:
- Jangan bagikan username dan password
- Ganti password jika dicurigai orang lain tahu
- Logout setelah selesai menggunakan aplikasi
